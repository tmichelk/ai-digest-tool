🔧 裏でAIがやっていた技術的な作業

📦 npm / Node.js

ツールを動かすエンジン。人間で言えば「体」の部分です。RSSの取得・AI呼び出し・LINE送信の処理を実行します。

📬 例え：荷物の仕分けをする倉庫作業員

📁 GitHub Actions

ツールを「3時間おきに自動で動かす」仕組み。コードをGitHubに置いておくだけで、指定した時間に自動実行してくれます。

⏰ 例え：スマホのアラーム機能。時間になったら自動で起こしてくれる

📡 RSS フィード

OpenAI・TechCrunch・Cursorなど15サイトのニュースを自動で取得する仕組み。サイトが新しい記事を出すと自動で受け取れます。

📰 例え：15社の新聞を毎日自動で配達してもらうサービス

🤖 Gemini API

Google のAIを呼び出す窓口。15サイト分の記事を渡すと「重要なものだけ選んで日本語で要約して」という指示を実行してくれます。

🧑‍💼 例え：優秀なリサーチャーに記事の要約をお願いする感じ

🔐 GitHub Secrets

APIキーなどの重要な情報を安全に保管する金庫。コードの中に直接書くと危険なので、別の場所に暗号化して保存します。

🔑 例え：パスワードをメモ帳に書かず、鍵のかかる金庫に入れる

💬 LINE Messaging API

プログラムからLINEにメッセージを送る仕組み。LINE公式アカウントを作り、トークルームに自動でメッセージを投稿できます。

📲 例え：ロボットがあなたのLINEに自動でメッセージを打ってくれる機能
